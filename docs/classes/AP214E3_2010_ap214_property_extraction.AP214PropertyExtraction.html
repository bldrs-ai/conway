Class AP214PropertyExtraction

Extracts STEP part properties from a populated AP214StepModel.

Walks the property chain general_property → (general_property_association) → property_definition → property_definition_representation → representation → descriptive_representation_item / measure_representation_item into per-part key/value rows. Includes the NIST validation properties (geometric/attribute validation property, e.g. volume via measure_representation_item).

Mirrors the IFC precedent (src/ifc/ifc_property_extraction.ts); feeds the web-ifc compat surface ap214_properties.ts that Share consumes.
